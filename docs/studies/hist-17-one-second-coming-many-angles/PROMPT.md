# Bible Study: One Second Coming, Many Angles (hist-17)

## Question
Do Revelation 1:7, 6:14-17, 14:14-20, and 19:11-21 describe the same event — the second coming of Christ? How do the winepress, cosmic signs, and universal visibility chains connect these passages? And is there biblical support for a separate "secret rapture" before the second coming?

## Methodology
This study follows the investigative methodology defined in `D:/bible/bible-studies/hist-series-methodology.md`. Evidence items must be registered in the evidence database.

**Source Restrictions:** No denominational writings.

```
INVESTIGATIVE METHODOLOGY:
- You are an investigator, not an advocate. Your job is to report what the evidence says.
- Gather evidence from ALL sides. If a passage is cited by historicists, examine it honestly. If a passage is cited by preterists, futurists, or idealists, examine it honestly.
- Do NOT assume your conclusion before examining the evidence.
- Do NOT state opinions. State what the text says. Do not use editorial characterizations like "genuine tension," "strongest argument," "most significant challenge," "honestly acknowledge," or "non-intuitive reading." Simply state what each passage says and what each side infers from it.
- When presenting findings, state: "The text says X" (explicit). Then state: "From this, Y interpretation infers Z" and "W interpretation infers V" (inferred).
- Never use language like "irrefutable," "obviously," or "clearly proves." Use "the text states," "this is consistent with."
- The conclusion should emerge FROM the evidence, not be imposed ON it.
```

**Key Findings from Related Studies:**

**From second-coming-revelation/CONCLUSION.md:**
- Revelation presents multiple descriptions of ONE pre-millennial second coming (Rev 1:7; 6:14-17; 11:15-18; 14:14-16; 16:15-21; 19:11-21), each with different emphasis
- The wrath/winepress chain (Rev 14:10 → 14:19 → 14:20 → 19:15 → 6:16-17) proves chapters 6, 14, and 19 describe one event
- The "all kindreds wail" chain (Rev 1:7 → 6:15-16) proves universal visibility = same event
- OT source unity: Dan 7:13, Zech 12:10, Isa 63:1-3, Joel 3:13 all feed into these passages
- "Thief" language (Rev 16:15; 2 Pet 3:10; 1 Thess 5:2) refers to timing, not visibility
- Greek erchomai used consistently for all "coming" statements; no vocabulary-based distinction
- Paul uses parousia, apokalypsis, and epiphaneia interchangeably (2 Thess 1:7-10; 2:1-8)
- "Every eye shall see him" (Rev 1:7) eliminates any "secret" coming

- 6th seal = FIRST VISION of second coming (wicked see Christ's face, "who can stand?")
