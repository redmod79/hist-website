## Revelation 13:2 -- Composite beast description
Greek Text: καὶ τὸ θηρίον ὃ εἶδον ἦν ὅμοιον παρδάλει, καὶ οἱ πόδες αὐτοῦ ὡς ἄρκου, καὶ τὸ στόμα αὐτοῦ ὡς στόμα λέοντος. καὶ ἔδωκεν αὐτῷ ὁ δράκων τὴν δύναμιν αὐτοῦ καὶ τὸν θρόνον αὐτοῦ καὶ ἐξουσίαν μεγάλην.

|------|-------|----------|---------|
| θηρίον | θηρίον | G2342 | N-NSN (beast) |
| ὅμοιον | ὅμοιος | G3664 | A-NSN (like/similar) |
| **παρδάλει** | **πάρδαλις** | **G3917** | **N-DSF (a leopard)** |

| στόμα | στόμα | G4750 | N-NSN (mouth) |
| ὡς στόμα | | | as a mouth |
| **λέοντος** | **λέων** | **G3023** | **N-GSM (of a lion)** |
| ἔδωκεν | δίδωμι | G1325 | V-AAI-3S (Aor Act Ind 3s, "gave") |
| ὁ δράκων | δράκων | G1404 | N-NSM (the dragon) |
| δύναμιν | δύναμις | G1411 | N-ASF (power) |
| **θρόνον** | **θρόνος** | **G2362** | **N-ASM (throne)** |
| ἐξουσίαν | ἐξουσία | G1849 | N-ASF (authority) |
| μεγάλην | μέγας | G3173 | A-ASF (great) |

**Key observation:** Rev 13:2 combines Daniel's four beasts in REVERSE order:
- παρδάλει (leopard) = Dan 7:6 (3rd beast)
- ἄρκου (bear) = Dan 7:5 (2nd beast)
- λέοντος (lion) = Dan 7:4 (1st beast)
- Dragon gives throne = Dan 7:7 (4th beast, from which the dragon inherits power)
